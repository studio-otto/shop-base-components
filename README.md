# Base Components
Studio Otto base components for Nuxt.js

## Install

Create `.npmrc` file in project directory
```
@studio-otto:registry=https://npm.pkg.github.com
```

Install package
```bash
yarn add @studio-otto/shop-base-components
```

## Developing
<details>
<summary>Publishing</summary>

- `npm version patch`
- `yarn build`
- `npm publish`
</details>
